## 🚀 Quick Start

```bash
# 1. Backend
cd ai-content-agent/backend
.\venv\Scripts\activate
python -m uvicorn main:app --host 0.0.0.0 --port 8002

# 2. Frontend (new terminal)
cd ai-content-agent/frontend
npm run dev

# 3. Open browser
http://localhost:5173
```
